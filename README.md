# 🎓 Student Management API

A **production-ready** Student Management System built with **FastAPI** + **PostgreSQL**, featuring clean architecture, soft delete, and complete CRUD operations.

[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

---

## 🚀 Features

- ✅ **CRUD Operations** - Create, Read, Update, Delete students
- ✅ **Soft Delete** - Active/Inactive status (data never lost)
- ✅ **Nested Address** - Complex nested models handling
- ✅ **Auto Age Calculation** - Computed from Date of Birth
- ✅ **Email Validation** - Domain validation + format check
- ✅ **Phone Validation** - 10-digit Indian mobile format
- ✅ **Clean Architecture** - Repository + Service pattern
- ✅ **PostgreSQL Database** - Production-grade database
- ✅ **JSON Fallback** - Switch between JSON/PostgreSQL storage
- ✅ **Swagger Docs** - Auto-generated API documentation

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Framework** | FastAPI |
| **Database** | PostgreSQL + SQLAlchemy ORM |
| **Validation** | Pydantic v2 |
| **Server** | Uvicorn |
| **Storage** | JSON (fallback) |
